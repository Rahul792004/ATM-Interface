#ATM Interface

## Project Title

ATM Interface

## Description

The ATM Interface is a Java console-based application that simulates basic ATM operations. Users can securely log in using a User ID and PIN to access banking functionalities such as withdrawal, deposit, transfer, and transaction history.

## Features

* User Login Authentication
* Transaction History
* Withdraw Money
* Deposit Money
* Transfer Money
* Balance Check
* Logout/Quit Option

## Technologies Used

* Java
* Object-Oriented Programming (OOP)

## Functionalities

### 1. Login System

* User enters User ID and PIN
* Validates user credentials

### 2. Transaction History

* Displays previous transactions performed by the user

### 3. Withdraw

* Withdraw money from account
* Checks available balance before withdrawal

### 4. Deposit

* Deposit money into account
* Updates balance automatically

### 5. Transfer

* Transfer money to another account
* Validates sufficient balance

### 6. Quit

* Exit the application safely

## How to Run the Project

1. Install Java (JDK)
2. Open project in any Java IDE (Eclipse/IntelliJ/VS Code)
3. Compile and run the `ATM.java` file
4. Enter User ID and PIN to access ATM features
